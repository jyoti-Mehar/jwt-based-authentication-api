# JWT Based Authentication API

A secure backend authentication system built using Node.js and Express with a simple frontend for testing.

## Features
- User Registration & Login
- Password hashing using bcrypt
- JWT Authentication
- Role-based access (Admin/User)
- Protected routes
- Error handling middleware

## Tech Stack
- Node.js
- Express.js
- MongoDB
- JWT
- bcrypt

## Project Structure
- config/ → DB connection
- controllers/ → logic
- middleware/ → auth & error handling
- models/ → database schema
- routes/ → API routes
- utils/ → helper functions
- auth-frontend/ → simple UI for testing

## How to Run
1. Clone repo
2. npm install
3. Create .env file
4. npm start
